Multimap is a fast 1:n key-value store that provides a mapping from keys to lists of values. This type of data structure is actually known as multimap, hence the name. Though most programming languages offer some kind of multimap standard container, this library is designed for amounts of data that do not fit into memory and/or which require external persistent storage.

Features

  • Embeddable store with a clean C++ and Java interface
  • Supported operations: put, get, remove, replace
  • Keys and values are arbitrary byte arrays
  • Keys are hold in memory, values are stored on disk
  • Import/export of Base64 key-value pairs from/to text files
  • Full thread-safe

Get Started

  • Read the overview and learn the basics.
  • Try the C++ or Java tutorial to get familiar with the API.

C++ Example

#include <multimap/Map.hpp>

int main() {
  multimap::Map::Options options;
  options.create_if_missing = true;
  multimap::Map map("path/to/directory", options);

  map.put("key", "1st value");
  map.put("key", "2nd value");

  auto iter = map.get("key");
  while (iter->hasNext()) {
    doSomething(iter->next());
  }

  // d'tor of iter releases internal reader lock.
  // d'tor of map flushes in-memory data to disk. 
}

Java Example

import io.multimap.*;

public static void main(String[] args) {
  Map.Options options = new Map.Options();
  options.setCreateIfMissing(true);
  Map map = new Map("path/to/directory", options);

  map.put("key", Utils.toByteArray("1st value"));
  map.put("key", Utils.toByteArray("2nd value"));

  Iterator iter = map.get("key");
  while (iter.hasNext()) {
    doSomething(iter.next());
  }

  iter.close();
  map.close();
}
